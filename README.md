# Most Recent Tab — Brave/Chrome Extension

Jump to your most recently used tabs with keyboard shortcuts.

Tired of clicking around to find the tab you were just on? Most Recent Tab lets you jump back instantly using keyboard shortcuts.

## Features

- Alt+Q — Jump to most recent tab
- Alt+W — Jump to 2nd most recent tab
- Alt+E — Jump to 3rd most recent tab
- Fully customizable shortcuts via `brave://extensions/shortcuts` or `chrome://extensions/shortcuts`
- Works across multiple windows
- Lightweight — no background bloat

## Installation

### Option 1 — Chrome Web Store
Search for **"Most Recent Tab"** on the Chrome Web Store and click Install.

### Option 2 — Load unpacked (Developer mode)
1. Download or clone this repository
2. Open `brave://extensions` or `chrome://extensions`
3. Enable **Developer mode** (top right toggle)
4. Click **"Load unpacked"**
5. Select this folder

## Setting your shortcuts

Default shortcuts are not pre-set. To configure them:
1. Go to `brave://extensions/shortcuts` or `chrome://extensions/shortcuts`
2. Find **Most Recent Tab**
3. Set your preferred keys for each action
